**Linguagem**: Python<br>
**Framework**: Flask<br>

**Dependências**
- Python 3.9.0 ou superior
- instalar as bibliotecas necessárias.
  
**Execução**
- Clone o projeto (ou baixe):
  
  ```shell

- Instale o framework necessário:
    ``` 
    pip install tkinter
